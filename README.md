# MERN driving app

Just a quick MERN web app.

This app has a data base hosted on mLab so it will start with data.

## Installing

To install clone this repo and run:

```
npm i
cd ./client
yarn
cd ../server
npm i
cd ../
```

## Running

```
npm run start
```

This should start the server and client and go to http://localhost:3000/

## Run tests

```
cd ./client
yarn test
```
